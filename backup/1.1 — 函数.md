## 初识函数
函数是数学中描述变量间依赖关系的核心概念,例如路程公式：
```math
s=vt
```
表示了路程、速度、和时间的关系。
```math
一般地，我们称这是v、t关于s的函数解析式。并且，我们用f(x)表示关于x的函数
```
## 函数图像
```math
现有一函数f(x)=x,为了画出图像，我们一般写为y=f(x)的形式，即y=x。
```
```math
画图像，就是在平面直角坐标系中,从x=-\infty到x=\infty,每个x对应的y等于x，再将每个点画出来。
```
